# Naming Conventions

- **Shell functions** (`functions/*.zsh`): snake_case or short lowercase (e.g., `obslink`, `chat`, `each`)
- **Scripts** (`scripts/`): kebab-case filenames (e.g., `obsidian-symlink.sh`, `start-day.sh`)
- **Aliases** (`aliases.zsh`): ≤6 chars, lowercase (e.g., `gs`, `gc`, `gp`, `v`)
- **Env vars**: UPPER_SNAKE_CASE (e.g., `OBSIDIAN_VAULT`, `PROJECTS_ROOT`)
- **Config files**: lowercase, match tool convention (e.g., `starship.toml`, `kitty.conf`)
